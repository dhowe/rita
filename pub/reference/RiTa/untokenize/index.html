Class

RiTa

Name

untokenize
Description Joins an array (of words and punctuation) into a sentence, according to the Penn Treebank conventions. The inverse of RiTa.tokenize().
Example 

  words = [ "The", "doctors", "treated", "dogs", "." ];
  sentence = RiTa.untokenize(words);
Parameters
String[]the input, an array of words (tokens) and punctuation
Stringdelimiter to use (optional, default=" " [a space])
Returns
Stringcontaining all elements as a sentence
Related RiTa.tokenize()
Note tmp_note
Syntax 
  RiTa.untokenize(arrayOfWords);
  RiTa.untokenize(arrayOfWords, delimiter);
Platform Java / JavaScript
